## Installation and Setup

1. Clone and navigate to the repository:

```bash
git clone git@github.com:hanzeelvilla/nest-files-example.git
cd nest-files-example
```

2. Install dependencies:

```bash
npm install
```

3. Rename the `.env.example` file to `.env` and update the environment variables with your R2 credentials.

4. Start the development server:

```bash
npm run dev:start
```

## API Endpoints

- `POST /api/files`: Upload a single file.
